## 3. Sistema de Branches

### 3.1 Estrutura de Branches

O plugin utiliza duas branches principais:

#### Branch `main`
- **Propósito:** Versões estáveis released
- **Política:** Apenas merges, nunca commits diretos
- **Histórico de Versões:**
  - 0.1.2 - Versão estável com FAQ
  - 0.2.0 a 0.2.5 - Versões com Mega Menu

#### Branch `develop`
- **Propósito:** Desenvolvimento de novas funcionalidades
- **Política:** Branch ativa de desenvolvimento
- **Fluxo:** Novas features → develop → teste → merge para main

### 3.2 Fluxo de Trabalho

```
1. Desenvolvimento (develop)
   │
   ├───> Commit incremental (v0.2.x-dev)
   │
   └───> Versionamento (v0.2.x)
         │
         └───> Merge para main (release)
               │
               └───> Push para remote
```

### 3.3 Comandos Git

```bash
# Ver branches locais
git branch

# Ver todas as branches (incluindo remote)
git branch -a

# Criar e mudar para nova branch de desenvolvimento
git checkout -b develop

# Fazer merge de develop para main
git checkout main
git merge develop

# Enviar para repositório remoto
git push origin main develop
```

## 4. Módulos do Plugin

### 4.1 Módulo FAQ (Versão 0.1.1+)

**Shortcode:** `[faq-patropi]`

**Funcionalidades:**
- Custom Post Type para FAQs
- Campo de seleção múltipla de páginas
- Accordion com opções personalizáveis
- Sistema de CSS customizável com editor e reset
- Toggle para ativar/desativar no admin
- Sistema de ícones com rotação de 180º

**Ícones disponíveis:**
- Insert, Remove
- Arrow Up, Arrow Down
- Arrow Up (Alt2), Arrow Down (Alt2)
- Plus (Alt2), Minus

---

### 4.2 Módulo Mega Menu (Versão 0.2.0+)

**Shortcode:** `[patropi-mega-menu]`

**Funcionalidades do Módulo:**

#### Menu Principal (Desktop)
- Interface gráfica no admin para criar menu (builder)
- Estilos customizáveis (cores, fundo, borda)
- Layout do menu: alinhamento (esquerda/centro/direita)
- Gap configurável entre itens
- Padding configurável dos itens (top/bottom e left/right)
- Transparência toggles para todas as cores
- Borda inferior configurável (espessura e cores)

#### Mega Menu (Dropdown)
- Largura configurável (ex: 100% ou 1200px)
- Largura máxima configurável
- Trigger: hover (ao passar mouse) ou click (ao clicar)
- Animação: FadeIn/FadeOut ou SlideIn/SlideOut
- Padding configurável do dropdown (top/bottom e left/right)
- Opção de sombra no dropdown
- Cor de fundo do dropdown (com opção transparente)

#### Colunas do Mega Menu
- 1-6 colunas configuráveis por item
- Largura em percentual por coluna
- Título opcional por coluna
- Dois layouts por coluna:
  - **Layout Links**: adicionar páginas ou links customizados (máx 10)
  - **Layout Imagem**: background cover, ícone, título, descrição, link

#### Mobile/Tablet
- Breakpoint em 1024px
- Botão hamburger para abrir menu
- Menu desliza da direita (slide)
- Overlay com fade
- Card de configurações separado no admin
- Cor do ícone configurável
- Cor de fundo configurável
- Largura do menu configurável (50-100%)
- Accordion com animação de slide
- Submenus empilhados verticalmente

## 5. Estrutura de Configurações (Banco de Dados)

O plugin salva as configurações na tabela `wp_options` com a chave `patropi_addon_settings`:

```php
$settings = array(
    'enabled' => 1,
    'main_menu' => array(
        'text_color' => '#333333',
        'text_hover' => '#2c3e50',
        'text_color_transparent' => false,
        'text_hover_transparent' => false,
        'bg_color' => 'transparent',
        'bg_hover' => 'transparent',
        'bg_color_transparent' => false,
        'bg_hover_transparent' => false,
        'border_width' => '0px',
        'border_color' => 'transparent',
        'border_hover' => '#2c3e50',
        'align' => 'flex-start',
        'item_gap' => '0px',
        'item_padding_y' => '15px',
        'item_padding_x' => '20px'
    ),
    'mega_menu' => array(
        'width' => '100%',
        'max_width' => '1200px',
        'trigger' => 'hover',
        'animation' => 'fade',
        'padding_y' => '20px',
        'padding_x' => '20px',
        'shadow' => false,
        'dropdown_bg' => '#ffffff',
        'dropdown_bg_transparent' => false,
        'mobile_icon_color' => '#333333',
        'mobile_bg_color' => '#ffffff',
        'mobile_width' => 85
    ),
    'menu_items' => array()
);
```
